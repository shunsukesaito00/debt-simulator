# 変更前後の計測（GA4）

## 目的

読みやすさ・情報設計の変更が、滞在やエンゲージメントに与える影響をざっくり確認する。

## 手順

1. **Google Analytics 4** の該当プロパティを開く。
2. **レポート** → **エンゲージメント** → **概要**（または **ページとスクリーン**）。
3. 次の指標を、**変更デプロイ前後2週間**など同じ長さの期間で比較する。
   - エンゲージメント率（`Engagement rate`）
   - 平均エンゲージメント時間（`Average engagement time`）
   - 主要ランディングページ（`/`、`/articles/*`）のセッション数・離脱
4. 比較は **探索** の「期間比較」、または **レポートの比較機能** を利用。

## 補助

- サイト内の `TrackedLink` イベント（`action` / `location`）で、CTA や記事リンクのクリック推移を参照できる場合は併記する。

## メモ

- 季節・キャンペーン・検索アルゴリズムなど外部要因でブレるため、**絶対評価より前後比較**を優先する。
