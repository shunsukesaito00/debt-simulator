# ビジュアル回帰確認（手動）

リリース前に、レイアウト崩れ・タイポ・ヘッダの折り返しをざっと確認するための URL 一覧です。

| 優先 | URL | 確認の目安 |
|------|-----|------------|
| 高 | `/` | ヒーロー、「最近の更新」ブロック、ピラー、フッタ |
| 高 | `/welcome` | はじめての方の導線・幅 |
| 高 | `/articles` | 一覧カード・セクション件数 |
| 高 | 任意の記事1本（例: `/articles/borrow-100-interest`） | `ArticlePageShell`・目次・本文幅 |
| 高 | `/simulator/cardloan` | フォーム・グラフ・余白 |
| 中 | `/tools/fixed-cost-impact` | ツール枠・結果表示 |
| 中 | `/income` | 収益一覧の表・カード |
| 低 | `/updates` | 更新ログの可読性 |

**ブレークポイント**: 狭い幅（iPhone 系）と 1280px 前後でヘッダナビの折り返しを確認（`.ds-container` は `max-w-6xl`）。
