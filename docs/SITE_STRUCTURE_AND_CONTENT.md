# サイト構造・記事内容の現状（外部共有・AIアドバイス用）

このドキュメントは、がいぶしゃ・他のAIにアドバイスをもらうために、現時点のサイト構造と記事内容を文字で書き起こしたものです。更新日時点のスナップショットとして参照してください。

## 0. 情報設計の更新（参考サイトの「パターン」のみ活用）

**方針**：外部サイトの文言・商標・独自表現の流用はしない。**ツール＋体験（シミュ・早見表・固定費ツール・条件別記事）**は差別化として維持する。

| 参考側のイメージ | 当サイトでの対応 |
|------------------|------------------|
| グローバルナビがシンプル（感情寄りの導線） | プライマリ：ホーム／はじめての方／記事／運営者プロフィール／お問い合わせ／プライバシー。シミュ・使い方・FAQ・計算ロジックは「ツール」ドロップダウン（PC）／モバイルは MENU 内サブメニュー |
| 「はじめての方」ランディング | **`/welcome`** … 推奨読む順・できること／できないこと・ツール導線 |
| トップのブロック順（共感→記事→自己紹介→テーマ別→定番…） | **`/`** を上記に近い順に再構成（中身は自サイト文案）。計測は `TrackedLink` の `location` でブロック単位 |
| 記事一覧の「体験記」枠 | **`/articles`** 先頭に「体験記・ストーリー」レール（`kind: story` または badge「体験記」） |
| WordPress 風の読みやすさ | 見出しにセリフ（Noto Serif JP）、本文・UI は Zen Kaku Gothic New。`.ds-blog-prose`／`.ds-article-shell` で本文幅を調整。**WordPress への移行はしない**（Next のまま） |
| モダンなプロダクトUI | 角丸 12px 前後・薄い影・ナビはテキストリンク＋アクティブ下線。CTA は `--ds-accent-strong`（緑）系。詳細は `app/globals.css` の `ds-*` |
| タグライン帯 | ヘッダ最上段に「悩んでいる方へ。一緒に考えていきましょう。」帯（`bg-emerald-950`）。ヒーロー冒頭にエコー |
| 記事ページの共通強化 | `ArticlePageShell` に日付+カテゴリバッジ。`ArticleToc`（折りたたみ目次）、`ArticleShareBar`（URLコピー）、`ArticlePrevNext`（前後記事ナビ）を新設 |
| 関連記事カード型 | `ArticleFooter` が `/articles/:slug` リンクを `getArticle()` で引きカード（タイトル+要約+バッジ）に展開 |
| フッター充実 | `FooterNav` を2カラム化。「サイト情報」列＋「カテゴリ」列（件数バッジ付き）＋タグライン |
| 一覧セクション件数 | `/articles` 各セクション見出しに件数バッジ表示 |

**真似しないもの**：文言のコピペ、ロゴ・画像の流用、商標に紛らわしい名称、ツール弱体化（参考サイトはツールが薄いため、当サイトはシミュ・固定費ツールを前面に残す）。

### 採用方針：参照サイトと「同型にしない」項目（意図的差分）

以下はパターン参考の対象外とし、**劣化させて揃えない**。

| 項目 | 理由 |
|------|------|
| **ツールの厚み** | シミュレーター・早見表・固定費インパクトは中核。参照元よりツールを前面に置く。 |
| **技術スタック** | Next.js App Router を維持。WordPress への移行はしない。 |
| **ブランド・独自コピー** | サイト名・文章は自サイトオリジナル。商標・独自表現の流用は禁止。 |
| **WP 付帯機能** | コメント欄・テーマ依存の固定レイアウト等は要件にない限り実装しない。 |

## 1. サイトの位置づけ・運営目的

- **サイト名**  
  借入返済シミュレーター（ドメイン・ブランド名はそのまま）

- **運営目的（about ページより）**  
  借入返済・固定費の見え方・支出の改善・返済計画などを、**条件別に比較・試算**し、判断材料を提供する。一般論ではなく**具体条件**で月々負担の違いを確認できるよう、シミュレーターと記事を用意し、**記事とツールを往復して判断**できる構成にしている。

- **主語の広げ方**  
  当初の「借入返済だけ」から、「毎月の固定負担を条件別に比較・改善する生活改善シミュレーションサイト」へ広げている。借入返済は残しつつ、固定費見直し・改善効果の試算・逆算・計画などを並列のテーマとして扱う。

- **信頼性**  
  金融系としての信頼感を維持。入力値はサーバーに送信せずブラウザ内のみで計算。注意書き・計算ロジック・FAQ・使い方ページを用意。

- `/how-to` … 使い方

- `/faq` … FAQ
- `/quick-reference` … 早見表（100万・200万・300万・3年/5年）
- `/privacy` … プライバシーポリシー
- `/contact` … お問い合わせ
- `/disclaimer` … 免責事項（存在する場合）

### ツール
- `/simulator/cardloan` … **カードローン返済シミュレーター**（中核ツール。借入額・金利・返済方式・追加返済・A/B比較・CSV出力）
- `/tools/fixed-cost-impact` … **固定費削減インパクト計算**（毎月削減額×継続月数で1年・3年・5年の累計を表示する軽量ツール）

### 記事
- `/articles` … 記事一覧（画面・メタの見出しは **悩み別に読む（記事一覧）**。パンくずラベルは `ARTICLES_INDEX_CRUMB_LABEL` と同一）

### ヘッダーナビのリンク
- **プライマリ**：ホーム / はじめての方（/welcome）/ 記事（/articles）/ 運営者プロフィール（/about）/ お問い合わせ（/contact）/ **プライバシー（/privacy）**

## 3. トップページの構成（上から順）

1. **ヒーロー** … 短いキャッチ・共感文案、`/welcome` とシミュへの CTA、早見表リンク  
2. **おすすめの記事** … `getHomeSpotlightArticles()`（編集キュレ・固定スラッグ）  
3. **最近の記事** … `getRecentArticles(5)`（`lib/articles.ts` の `publishedAt` 降順。未設定記事はリストに出ない）  
4. **運営者一言** … 引用風カード＋ `/about`（運営者プロフィール）  
5. **悩み別に読む（テーマピラー）** … 記事一覧アンカー＋シミュ＋固定費ツールの6前後  
6. **よく読まれている記事** … `getPopularArticles()`（手動キュレーション）  
7. **はじめての方へ（3カード）** … 読む順・シミュ・体験記カテゴリへの導線  
8. **お問い合わせ** … `/contact`（環境変数があれば note / X の外部リンクを併記）  
9. **このサイトについて** … 短文＋記事一覧・使い方・FAQ・ロジック・プライバシー  
10. **注意事項** … 参考情報・契約優先など短文

## 4. 記事一覧ページ（/articles）の構造

- **タイトル（画面・SEO）**  
  **悩み別に読む（記事一覧）**（メタ `title` は `｜借入返済シミュレーター` 付きでキーワード維持）

- **導入文**  
  悩み別に読むトーン。体験記→解説→シミュの順でもよい、という導線。早見表へのリンクあり。  

- **体験記・ストーリー（先頭ブロック）**  
  `getStoryArticles()`（`kind: "story"` または badge「体験記」）をカード表示。`publishedAt` があれば日付表示。  

- **セクション構成（6ブロック）**  
  記事一覧は「表示用セクション」と「記事カテゴリ」を対応させて表示。各セクションは id でアンカーリンク可能。ラベル・説明は `lib/articles.ts` の `ARTICLE_LIST_SECTIONS` が正（以下は概要）。

  | セクション id | ラベル（表示） | 表示される記事カテゴリ（内部） |
  |---------------|----------------|--------------------------------|
  | fixed-cost | 固定費をゆるく見直す | fixed-cost |
  | household | 家計の見え方・体験記 | household |
  | improvement-effect | 少し削ったらどう変わる？ | improvement-effect |
  | loan-comparison | 借入・リボの負担を比べる | loan-amount, repayment-method, revolving |
  | repayment-improvement | 返済を軽くするヒント | repayment-improvement |
  | repayment-planning | 返せる額から考える | repayment-planning |

- **記事が0件のセクション**  
  「このカテゴリの記事は準備中です。順次追加していきます。」と表示。

- **下部CTA**  
  「自分の条件で試算する」：返済額を試算する（シミュレーター）→ / 固定費削減インパクトを計算する →

## 5. 記事カテゴリと記事一覧（slug / タイトル / カテゴリ）

## 7. 記事の共通構造（people-first テンプレート）

## 8. 主要記事の内容要約（検索意図・条件・結論のイメージ）

- **borrow-100-interest**  
  借金100万円・年利15%・元利均等・3年/5年比較。3年なら総利息約24.8万・毎月約34,665円、5年なら総利息約42.7万・毎月約23,790円。月々を軽くすると総支払額は増えるトレードオフを説明。

- **borrow-200-monthly-payment**  
  借金200万円・年利15%・3年/5年。3年なら月々約6.9万・総利息約49.6万、5年なら月々約4.8万・総利息約85.5万。200万クラスでの期間選択の影響の大きさを説明。

- **fixed-payment-principal-interest-cannot-payoff**  
  100万・年利15%・定額元利で、毎月1万・1.5万・3万の3パターン比較。1万は利息に届かず完済不可に近い、1.5万は完済するが約154か月・総利息約131万、3万なら約50か月・総利息約47.5万。返済額が利息を上回ること・元本が十分減る水準の重要性を説明。

- **monthly-50000-interest-at-15percent**  
  毎月5万円返済・年利15%を固定し、借入100万・200万・300万で総利息・完済期間を比較。100万は約24か月・総利息約15.8万、200万は約56か月・約79万、300万は約112か月・約257.9万。同じ月5万でも借入額が増えると負担が急増することを説明。

- **early-repayment-100k-effect**  
  200万・年利15%・5年返済の例で、12か月後に10万円繰り上げした場合。完済約4か月短縮・総利息約12.9万円減。10万の繰り上げでも効果があること、総支払の見え方（前倒しで払う代わりに利息が減る）を説明。

- **fixed-cost-guide**  
  固定費見直しカテゴリの親記事。通信費・サブスク・保険など毎月の固定負担を見直すときの「何から手をつけるか」「改善効果を比較する」考え方を整理。固定費削減インパクト計算・借入返済シミュ・記事一覧への導線あり。

- **monthly-30000-how-much-can-borrow**  
  月3万円返済・年利15%で、3年なら約90万・5年なら約126万・7年なら約154万が借入額の目安。逆算の考え方と総利息の増え方を説明。

- **loan-amount-guide**  
  借入額別カテゴリの親記事。100万・200万・300万で返済負担がどう変わるかを整理し、このカテゴリで読むべき記事へのリンクをまとめる。

- **repayment-improvement-guide**  
  返済改善の親記事。繰り上げ返済・返済期間・毎月返済額・返済方式の見直しの4視点で「返済を軽くする方法」を整理。比較表とカテゴリ内記事へのリンクあり。

## 9. ツールの仕様

### カードローン返済シミュレーター（/simulator/cardloan）
- **入力**  
  借入金額、返済開始年月、返済方式（元利均等・元金均等・定額元利・定額元金）、返済期間 or 毎月返済額 or 毎月元金、金利（複数ステップ可）、追加返済（毎月/単発/ボーナス）。A/B 2案をタブ切替で比較。
- **出力**  
  毎月返済額・総返済額・利息合計・完済時期、グラフ、返済表、A/B比較表。CSV ダウンロード可。
- **その他**  
  条件に連動した「あわせて読みたい」記事、返済改善ブロック、注意文・計算ロジック・FAQ へのリンク。GA4 計測付き。

### 固定費削減インパクト計算（/tools/fixed-cost-impact）
- **入力**  
  毎月の削減額（円）、継続月数（任意。指定時はその期間の累計も表示）。
- **出力**  
  1年（12か月）・3年（36か月）・5年（60か月）の累計削減額。指定した継続月数がある場合はその累計も表に表示。計算は「毎月の削減額 × 継続月数」のみ。
- **導線**  
  固定費見直しの進め方、借入返済シミュレーター、記事一覧へのリンク。

---

## 10. 導線の流れ（簡潔に）

- **トップ** → はじめての方（/welcome）/ おすすめ記事 / テーマピラー / 定番記事 / シミュ・固定費ツールへの導線
- **はじめての方** → 推奨記事順・シミュ・固定費ツール・注意（投資助言ではない等）
- **記事一覧** → 各カテゴリセクション（アンカー）・記事カード・下部でシミュレーターと固定費インパクト
- **個別記事** → ArticleFooter でシミュレーター CTA・関連記事・カテゴリ一覧・記事一覧へ戻る
- **固定費見直し** → 固定費ガイド記事から固定費削減インパクト・シミュ・記事一覧
- **シミュレーター** → 条件連動の「あわせて読みたい」・返済改善ブロック・計算ロジック・FAQ 等

## 11. 技術・デザイン上の前提

- **フレームワーク**  
  Next.js（App Router）。主要ツール・記事は Client Component で state を保持。

- **計算**  
  借入返済は **lib/loan-calc.ts**（calcLoan）。固定費インパクトはフロントのみで乗算。

- **データ**  
  記事メタデータは **lib/articles.ts** の articlesData。カテゴリ・一覧セクション・関連記事・条件連動スコアもここで管理。

- **デザイン**  
  Tailwind。rounded-2xl/3xl、border-gray-200、bg-gray-50/80、font-black 見出しなどで統一。モバイルではボタン・表の横スクロールを考慮。

- **計測**  
  GA4。TrackedLink で action / location / target / link_type 等を送信。シミュレーター CTA・記事関連・カテゴリ入口・ツールカードなどを計測。
