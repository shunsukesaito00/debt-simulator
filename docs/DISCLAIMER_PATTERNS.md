# 免責・注意書きの表現パターン（金融コンテンツ）

**目的**: [`docs/ROADMAP.md`](./ROADMAP.md) の運用方針に沿い、各ページで使う定型句の候補を一箇所にまとめる。

## 試算・ツール全般

- 「本ツール／本記事の試算は**参考・概算**です。契約内容（金利・返済日・端数・手数料等）を優先してください。」
- 「計算は**一般的な月次の近似**です。金融機関の計算と差が出る場合があります。」

## 返済負担率・手取り（シミュレーター）

- 「返済負担率は**審査基準や適正な返済能力の判断を示すものではありません**。家計の目安用です。」

## 無利息期間（シミュレーター）

- 「無利息期間は**先頭Nヶ月の利息を0とする簡易モデル**です。実際のキャンペーン条件や元利均等の再計算は反映していません。」

## 記事（条件固定の数値例）

- 「数値は**条件を固定した例**です。自分の契約・商品では異なります。」
- 「**投資助言・借入勧誘**を目的としたものではありません。」（体験記・副業記事で必要に応じて）
